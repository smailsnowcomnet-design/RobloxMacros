================================================================================
                          БЫСТРОЕ НАЧАЛО (2 МИНУТЫ)
================================================================================

УСТАНОВКА И ЗАПУСК:
  1. Убедитесь, что установлен .NET 8.0
     (скачать: https://dotnet.microsoft.com/download)

  2. Откройте командную строку / PowerShell в этой папке

  3. Команды для сборки и запуска:
     dotnet build -c Release      (собрать)
     dotnet run                    (запустить)
     
     ИЛИ просто двойной клик на RobloxMacro.exe

ИСПОЛЬЗОВАНИЕ:
  1. Выбрать режим игры (нажать на карточку)
  2. Нажать "Начать запись"
  3. Выполнить действия в Roblox
  4. Нажать "Стоп запись"
  5. Нажать "Запуск повтора"

ГОРЯЧИЕ КЛАВИШИ (работают всегда):
  F7  → Быстрой запуск/стоп макроса
  F8  → Экстренная остановка

================================================================================
                           РЕШЕНИЕ ПРОБЛЕМ
================================================================================

ПРОБЛЕМА: Приложение не запускается
РЕШЕНИЕ: Проверьте версию .NET (dotnet --version должна быть 8.0+)

ПРОБЛЕМА: Горячие клавиши не работают
РЕШЕНИЕ: Убедитесь, что окно активно при первом запуске

ПРОБЛЕМА: Макрос движется неправильно
РЕШЕНИЕ: Нажмите "Настроить камеру Roblox" перед использованием

ПРОБЛЕМА: Что-то пошло не так
РЕШЕНИЕ: Нажмите F8 (экстренная остановка)

Подробнее читайте в README.md → Решение проблем

Просмотр версии .NET:
  dotnet --version

  dotnet restore

  dotnet build -c Debug

Сборка в режиме Release (оптимизировано):
  dotnet build -c Release

Запуск приложения:
  dotnet run

Публикация как самостоятельное приложение:
  dotnet publish -c Release -r win-x64 --self-contained

Очистка артефактов сборки:
  dotnet clean
