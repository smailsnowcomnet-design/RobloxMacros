# 🎮 Roblox Macro - Anime Vanguards Edition

Продвинутое приложение-макрос для автоматизации действий в игре Roblox (режим Anime Vanguards) на C# с использованием WPF (.NET 8).

## ✨ Возможности

### 1. **Интерфейс и выбор режимов**
- Современный UI с темной темой (Dark Theme / Fluent Design)
- Стильные карточки-кнопки для выбора игровых режимов:
  - 📖 Story Mode
  - 👑 Legends Stage
  - ♾️ Infinity Mode
  - ⚔️ Raid
  - ⭐ Challenge
- Информационная панель (Status Bar) с отображением выбранного режима
- Визуальные индикаторы статуса (запись, воспроизведение)

### 2. **Запись и воспроизведение макросов**
- **Запись**: Перехват и логирование:
  - Координаты кликов мыши
  - Нажатия клавиш клавиатуры
  - Временные задержки между действиями
- **Воспроизведение**: Циклическое повторение записанных действий
- Поддержка настройки количества повторений или бесконечного повтора
- Сохранение/загрузка макросов в формате JSON

### 3. **Windows API для точной симуляции**
- Использование `SendInput` для обхода стандартных ограничений окон
- Поддержка:
  - Движений мыши
  - Кликов левой/правой кнопки мыши
  - Нажатий и отпусканий клавиш
  - Комбинаций клавиш (Ctrl+A, Shift+C и т.д.)
  - Ввода текста через клавиатуру

### 4. **Настройка камеры Roblox**
- Специальная кнопка для автоматического выравнивания обзора
- Жесткая последовательность команд для стабилизации камеры:
  - Имитация правой кнопки мыши
  - Перемещение в центр экрана
  - Нажатие клавиш сброса ('O', 'I')
- Предотвращает сбои макроса из-за случайного сдвига экрана

### 5. **Глобальные горячие клавиши**
- **F7** — Запустить/остановить воспроизведение макроса
  - Работает даже когда окно приложения свернуто
  - Удобно для быстрого управления во время игры
- **F8** — Экстренная остановка (Panic Button)
  - Немедленно прерывает все действия
  - Полезна для чрезвычайных ситуаций

## 📋 Системные требования

- **OS**: Windows 10/11
- **.NET Runtime**: .NET 8.0 или выше
- **RAM**: 512 MB (минимум)
- **Разрешение экрана**: 1024x768 или выше (рекомендуется 1920x1080)

## 📖 Инструкция по использованию

### Основные шаги

1. **Запустить приложение**
   ```
   RobloxMacro.exe
   ```

2. **Выбрать режим игры**
   - Нажать на одну из карточек режимов (Story Mode, Legends Stage и т.д.)
   - Выбранный режим отобразится в информационной панели

3. **Записать макрос**
   - Нажать кнопку "▶ Начать запись"
   - Выполнить необходимые действия в игре Roblox
   - Нажать кнопку "⏹ Стоп запись"
   - Записанные действия отобразятся в счетчике

4. **Настроить параметры воспроизведения**
   - Установить количество повторений в поле "Повторов:"
   - Или включить чекбокс "♾️ Бесконечный повтор" для бесконечного цикла

5. **Запустить макрос**
   - Нажать кнопку "▶ Запуск повтора"
   - Макрос начнет автоматическое повторение
   - Нажать "⏹ Стоп повтор" для остановки

### Горячие клавиши (работают в любой момент)

```
F7  → Запустить/остановить воспроизведение
F8  → Экстренная остановка (Panic Button)
```

### Дополнительные функции

#### Настройка камеры Roblox
```
1. Нажать кнопку "📷 Настроить камеру Roblox"
2. Приложение автоматически выполнит последовательность команд:
   - Зажмет правую кнопку мыши
   - Переместит курсор в центр экрана
   - Нажмет клавишу 'O' для сброса
3. Камера выровняется для стабильности макроса
```

#### Сохранение профилей
```csharp
// Пример сохранения макроса в профиль
var manager = new MacroProfileManager();
manager.SaveProfile(
    profileName: "My Raid Macro",
    gameMode: GameMode.Raid,
    actions: recordedActions,
    description: "Оптимизированный макрос для Raid режима",
    repetitions: 10,
    infiniteRepeat: false
);

// Загрузка профиля
var profile = manager.LoadProfile("My Raid Macro");
if (profile != null)
{
    player.SetMacro(profile.Actions);
}
```

## 💻 Примеры кода

### Использование InputSimulator

```csharp
// Получить текущую позицию мыши
InputSimulator.GetCursorPosition(out int x, out int y);
Console.WriteLine($"Мышь на позиции: {x}, {y}");

// Переместить мышь
InputSimulator.MoveMouse(500, 300);

// Клик левой кнопкой мыши
InputSimulator.LeftClick();

// Клик левой кнопкой в конкретной позиции
InputSimulator.LeftClickAt(600, 400);

// Нажать клавишу
InputSimulator.KeyClick(InputSimulator.VirtualKeys.VK_A);

// Комбинация клавиш (Ctrl+S)
InputSimulator.KeyCombination(
    InputSimulator.VirtualKeys.VK_CONTROL,
    InputSimulator.VirtualKeys.VK_S
);

// Ввести текст
InputSimulator.TypeText("Hello World");
```

### Запись и воспроизведение

```csharp
// Инициализировать рекордер
var recorder = new MacroRecorder();

// Начать запись
recorder.StartRecording();

// ... выполнить действия ...

// Остановить запись
recorder.StopRecording();

// Сохранить в файл
recorder.SaveToFile("my_macro.json");

// Инициализировать плеер
var player = new MacroPlayer();

// Загрузить макрос
player.LoadFromFile("my_macro.json");

// Подписаться на события
player.StatusChanged += (status) => Console.WriteLine(status);
player.PlaybackCompleted += () => Console.WriteLine("Завершено!");

// Запустить воспроизведение (5 повторений)
player.StartPlayback(5);

// Или бесконечное воспроизведение
player.StartPlayback(0);

// Остановить воспроизведение
player.StopPlayback();
```

### Регистрация горячих клавиш

```csharp
var hotkeyManager = new HotkeyManager(windowHandle);

// Зарегистрировать F7
hotkeyManager.RegisterHotkey(
    keyCode: (uint)InputSimulator.VirtualKeys.VK_F7,
    action: () => Console.WriteLine("F7 нажат!"),
    modifiers: HotkeyModifier.None
);

// Зарегистрировать Ctrl+Alt+M
hotkeyManager.RegisterHotkey(
    keyCode: (uint)InputSimulator.VirtualKeys.VK_M,
    action: () => Console.WriteLine("Ctrl+Alt+M нажат!"),
    modifiers: HotkeyModifier.Ctrl | HotkeyModifier.Alt
);

// Очистить все горячие клавиши при завершении
hotkeyManager.Dispose();
```

## ⚙️ Виртуальные коды клавиш

Доступные константы в `InputSimulator.VirtualKeys`:

## 🐛 Решение проблем

### Проблема: Приложение не перехватывает горячие клавиши
**Решение**: 
- Убедитесь, что окно приложения активно при первом запуске
- Перезапустите приложение
- Проверьте, что никакое другое приложение не использует F7/F8

### Проблема: Макрос выполняется медленнее, чем ожидалось
**Решение**: 
- Уменьшите задержки между действиями в записи
- Используйте DirectInput вместо стандартного ввода в Roblox настройках
- Закройте ненужные приложения в фоне

### Проблема: Мышь не движется на правильную позицию
**Решение**: 
- Проверьте разрешение экрана (приложение записывает абсолютные координаты)
- Убедитесь, что Roblox окно находится в том же положении при воспроизведении
- Используйте функцию "Настроить камеру" перед записью

### Проблема: Клавиши не вводятся правильно
**Решение**: 
- Убедитесь, что язык ввода установлен на английский
- Проверьте, что окно Roblox активно при воспроизведении
- Попробуйте использовать комбинацию клавиш вместо отдельных нажатий
