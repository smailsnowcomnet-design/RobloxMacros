Перед началом убедитесь:

1. .NET 8.0 установлен?
   
   Откройте PowerShell и выполните:
     dotnet --version
   
   Должно показать: 8.0.xxx или выше
   
   Если не установлено: скачайте с https://dotnet.microsoft.com/download/dotnet/8.0

2. Файлы в нужном месте?
   
   Текущая папка должна содержать:
     ✅ RobloxMacro.csproj
     ✅ Папка Models/
     ✅ Папка Services/
     ✅ Папка ViewModels/
     ✅ *.md файлы документации
   
   Проверьте командой:
     ls RobloxMacro.csproj

3. Готово к сборке?
   
   Выполните:
     dotnet build -c Release
   
   Должно показать:
     ✅ Build succeeded

ЭТАП 1: ЧТЕНИЕ (5-10 минут)
  
  1. Прочитайте: START.md (этот файл)
  2. Прочитайте: SUMMARY.txt (краткое резюме)
  3. Выберите:   QUICKSTART.md ИЛИ README.md

ЭТАП 2: УСТАНОВКА (2-5 минут)
  
  1. Проверьте: dotnet --version
  2. Если нужно: Установите .NET 8.0 с https://dotnet.microsoft.com/download
  3. Перейдите: В папку проекта (где RobloxMacro.csproj)

ЭТАП 3: СБОРКА (2-3 минуты)
  
  1. Откройте: PowerShell или Command Prompt
  2. Выполните: dotnet build -c Release
  3. Проверьте: Должно написать "Build succeeded"

ЭТАП 4: ЗАПУСК (1 минута)
  
  1. Выполните: dotnet run
  2. Должно появиться: Окно приложения с темным интерфейсом
  3. Готово! 🎉

ЭТАП 5: ИСПОЛЬЗОВАНИЕ (по мере надобности)
  
  1. Выберите: Режим игры
  2. Запишите: Макрос (кнопка "Начать запись")
  3. Воспроизведите: Макрос (кнопка "Запуск повтора")
  4. Используйте: Горячие клавиши F7/F8

Q: Как быстро начать?
A: Откройте QUICKSTART.md и следуйте инструкциям (5 минут).

Q: Как установить .NET?
A: Скачайте с https://dotnet.microsoft.com/download/dotnet/8.0
   Запустите установщик, следуйте инструкциям, перезагрузитесь.

Q: Как собрать проект?
A: Откройте PowerShell и выполните: dotnet build -c Release

Q: Как запустить приложение?
A: Выполните: dotnet run

Q: Что делать, если ничего не работает?
A: Откройте BUILD_GUIDE.md или README.md (раздел "Решение проблем")

Q: Какие горячие клавиши?
A: F7 = запуск/стоп, F8 = экстренная остановка

Q: Где сохраняются макросы?
A: В %APPDATA%\RobloxMacro\Profiles\ в виде JSON файлов

Q: Можно ли изменять код?
A: Да, проект под MIT лицензией (см. LICENSE файл)
