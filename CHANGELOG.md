# CHANGELOG

## [1.0.0] - 2026-05-23

### ✨ Добавлено
- **Основной функционал**
  - ✅ Запись макросов с отслеживанием движений мыши и нажатий клавиш
  - ✅ Воспроизведение макросов с поддержкой кратного повтора
  - ✅ Бесконечный режим повтора
  - ✅ Сохранение и загрузка макросов в JSON формате

- **Игровые режимы**
  - ✅ Story Mode (📖)
  - ✅ Legends Stage (👑)
  - ✅ Infinity Mode (♾️)
  - ✅ Raid (⚔️)
  - ✅ Challenge (⭐)

- **Интерфейс (UI/UX)**
  - ✅ Современный темный интерфейс (Dark Theme / Fluent Design)
  - ✅ Скругленные углы и эффекты тени
  - ✅ Информационная панель (Status Bar)
  - ✅ Визуальные индикаторы статуса (запись/воспроизведение)
  - ✅ Карточки-кнопки для выбора режимов
  - ✅ Красивые элементы управления (кнопки, чекбоксы, текстовые поля)

- **Горячие клавиши**
  - ✅ F7 - Запустить/остановить воспроизведение
  - ✅ F8 - Экстренная остановка (Panic Button)
  - ✅ Работают глобально (даже когда приложение свернуто)

- **Специальные функции**
  - ✅ Настройка камеры Roblox (выравнивание обзора)
  - ✅ Система профилей для сохранения макросов
  - ✅ Сервис управления профилями

- **Windows API Интеграция**
  - ✅ SendInput для точной симуляции ввода
  - ✅ GetCursorPos / SetCursorPos для управления курсором
  - ✅ RegisterHotKey для глобальных горячих клавиш
  - ✅ Поддержка всех стандартных виртуальных кодов клавиш

- **Архитектура**
  - ✅ MVVM паттерн для отделения UI от логики
  - ✅ Многопоточность для неблокирующей записи/воспроизведения
  - ✅ События и уведомления для реактивного обновления UI
  - ✅ Модульная структура кода

- **Документация**
  - ✅ README с полной инструкцией
  - ✅ QUICKSTART для быстрого начала
  - ✅ API_REFERENCE с документацией всех методов
  - ✅ BUILD_GUIDE с инструкциями по сборке
  - ✅ ARCHITECTURE с описанием архитектуры

### 📋 Технические детали
- **Язык**: C# (.NET 8.0)
- **UI Framework**: WPF (Windows Presentation Foundation)
- **ОС**: Windows 10/11
- **Архитектура**: MVVM + Services

## Версионирование

Приложение использует семантическое версионирование (SemVer):
- **MAJOR** (1.x.x) - Крупные изменения, не совместимые с предыдущими версиями
- **MINOR** (x.1.x) - Новые функции, совместимые с предыдущими версиями
- **PATCH** (x.x.1) - Исправления ошибок
