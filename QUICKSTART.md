# Roblox Macro - Быстрый старт

## 🚀 За 5 минут до первого макроса

### Шаг 1: Запуск (30 сек)
```
1. Двойной клик на RobloxMacro.exe
2. Приложение запустится с темной темой
```

### Шаг 2: Выбор режима (30 сек)
```
1. Кликнуть на нужный режим (например, "Story Mode")
2. Режим отобразится в нижней панели
```

### Шаг 3: Запись (2 мин)
```
1. Нажать "▶ Начать запись"
2. Перейти в окно Roblox
3. Выполнить нужные действия (клики, нажатия клавиш)
4. Вернуться в приложение
5. Нажать "⏹ Стоп запись"
```

### Шаг 4: Воспроизведение (30 сек)
```
1. Установить количество повторений (или включить бесконечный повтор)
2. Нажать "▶ Запуск повтора"
3. Макрос начнет автоматическое повторение
```

## ⌨️ Быстрые команды

```
F7  → Быстрой запуск/стоп макроса (даже если приложение свернуто)
F8  → Экстренная остановка (Panic Button)
```

## 💡 Полезные советы

### Совет 1: Аккуратная запись
- Записывайте медленнее и плавнее
- Избегайте резких движений мыши
- Делайте паузы между действиями

### Совет 2: Стабильность
- Нажмите "📷 Настроить камеру Roblox" перед каждой игровой сессией
- Оставляйте окно Roblox на одном месте
- Используйте одинаковое разрешение экрана

### Совет 3: Оптимизация
- Экспортируйте часто используемые макросы в профили
- Используйте разные макросы для разных режимов
- Проверяйте макрос на малом количестве повторений сначала

### Совет 4: Безопасность
- Всегда используйте F8, если что-то идет не так
- Тестируйте макросы с минимальным количеством повторений
- Держите рядом мышь для ручного контроля

## ❓ Частые вопросы

**Q: Может ли приложение быть заметно при использовании?**
A: Нет! Используйте F7/F8 чтобы управлять макросом, не развернув приложение.

**Q: Как сохранить мой макрос?**
A: Профили автоматически сохраняются в %APPDATA%\RobloxMacro\Profiles\

**Q: Могу ли я использовать несколько макросов?**
A: Да! Просто записывайте разные макросы и переключайтесь между ними.

**Q: Какова максимальная скорость макроса?**
A: Ограничена только скоростью вашего компьютера и игры Roblox.

**Q: Будет ли мой аккаунт забанен?**
A: Используйте ответственно. Читайте условия Roblox.

## 🔧 Настройка для разных режимов

### Story Mode
- Медленнее, больше кликов
- Рекомендуется 5-10 повторений
- Сосредоточьтесь на кликах

### Legends Stage
- Быстрее, требует точности
- Используйте комбо клавиш
- Рекомендуется 20-50 повторений

### Infinity Mode
- Нужна скорость и выносливость
- Включите бесконечный повтор
- Будьте готовы к F8!

### Raid
- Стратегический режим
- Записывайте целую тактику
- 1-3 повторения обычно достаточно

### Challenge
- Самый сложный режим
- Требует идеальной синхронизации
- Тестируйте на малых повторениях
