## 🎯 Первый макрос за 2 минуты

### Шаг 1: Запустить приложение
```bash
dotnet run
# ИЛИ двойной клик на RobloxMacro.exe
```

### Шаг 2: Выбрать режим
```
Нажать на одну из карточек режимов
(Story Mode, Legends Stage и т.д.)
```

### Шаг 3: Записать
```
1. Нажать "▶ Начать запись"
2. Перейти в Roblox и выполнить действия
3. Вернуться в приложение
4. Нажать "⏹ Стоп запись"
```

### Шаг 4: Воспроизвести
```
1. Установить количество повторений
2. Нажать "▶ Запуск повтора"
3. Макрос начнет работать!
```

## 🐛 Часто встречающиеся проблемы

### Проблема: Не запускается
**Решение:**
```bash
# Проверить версию .NET
dotnet --version

# Должно быть 8.0 или выше
# Если нет - скачать с https://dotnet.microsoft.com/download
```

### Проблема: Горячие клавиши не работают
**Решение:**
- Убедитесь, что окно активно при первом запуске
- Перезагрузитесь
- Проверьте, не используются ли F7/F8 в других приложениях

### Проблема: Макрос движется неправильно
**Решение:**
- Нажмите "📷 Настроить камеру Roblox"
- Используйте ту же машину и разрешение экрана
- Держите окно Roblox в одном месте

**Больше → [README.md - Решение проблем](README.md#решение-проблем)**
