## 🎯 **Основные компоненты**

### 1. InputSimulator (Windows API)
```csharp
// Симуляция ввода на уровне системы
SendInput() → SendInput() → SimulateInput()
GetCursorPos() → SetCursorPos() → MoveMouse()
KeyDown() → KeyUp() → KeyClick()
```

### 2. MacroRecorder (Запись)
```csharp
// Мониторинг ввода в фоновом потоке
MonitorInputLoop() → GetCursorPos() → RecordAction()
SaveToFile() → JSON сохранение
```

### 3. MacroPlayer (Воспроизведение)
```csharp
// Асинхронное выполнение действий
PlaybackLoop() → ExecuteMacroActions() → ExecuteAction()
SendInput() → SendInput() → Выполнение
```

### 4. HotkeyManager (Горячие клавиши)
```csharp
// Глобальная регистрация клавиш
RegisterHotKey() → HwndHook() → ProcessHotkey()
F7 → TogglePlayback()
F8 → PanicStop()
```

### 5. MacroProfileManager (Профили)
```csharp
// Управление файлами макросов
SaveProfile() → JSON
LoadProfile() → JSON → MacroProfile
GetAvailableProfiles() → List<string>
```
