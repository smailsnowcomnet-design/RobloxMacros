# ✨ Полный список функций Roblox Macro

## 🎯 Основной функционал

### ✅ Запись макросов
- **Что это делает**: Записывает все ваши действия в игре (клики, нажатия клавиш, движения мыши)
- **Как использовать**:
  1. Нажать кнопку "▶ Начать запись"
  2. Выполнить нужные действия в Roblox
  3. Нажать кнопку "⏹ Стоп запись"
- **Что получается**: Записанный макрос готов к воспроизведению

### ✅ Воспроизведение макросов
- **Что это делает**: Автоматически повторяет записанные действия
- **Как использовать**:
  1. Установить количество повторений
  2. Нажать "▶ Запуск повтора"
  3. Макрос автоматически выполняется
- **Параметры**: 
  - Количество повторений (1-999)
  - Бесконечный повтор (♾️)

### ✅ Точная симуляция ввода
- **Мышь**:
  - Движение курсора
  - Левый клик
  - Правый клик
  - Зажатие и отпускание кнопок
- **Клавиатура**:
  - Нажатие любых клавиш
  - Отпускание клавиш
  - Комбинации (Ctrl+A, Shift+C и т.д.)
  - Ввод текста посимвольно
- **Задержки**: Точная синхронизация между действиями

## 🎨 Интерфейс (UI Features)

### 🌙 Темная тема (Dark Theme)
- Удобна для длительного использования
- Система цветов на основе Fluent Design
- Основной цвет: Синий (#0078D4)
- Фон: Темно-серый (#1E1E1E)

### 🎯 Выбор режимов
- **Визуальные карточки**: Каждый режим имеет свою карточку с иконкой
- **Быстрое переключение**: Один клик для выбора
- **Индикатор**: Выбранный режим отображается в информационной панели

### 📊 Информационная панель (Status Bar)
- **Текущий статус**: Показывает, что сейчас происходит
- **Индикаторы**: 
  - 🔴 Запись активна
  - 🟢 Воспроизведение активно
- **Счетчики**: Количество записанных действий
- **История**: Последние действия

### 🎛️ Панель управления
- **Кнопки**:
  - "▶ Начать запись" (зеленая)
  - "⏹ Стоп запись" (красная)
  - "▶ Запуск повтора" (оранжевая)
  - "⏹ Стоп повтор" (красная)
- **Визуальная обратная связь**: Кнопки светятся при наведении

### ⚙️ Параметры
- **Количество повторений**: Текстовое поле с проверкой значения
- **Бесконечный повтор**: Чекбокс для циклического режима

### 🔧 Дополнительные функции
- **Кнопка "📷 Настроить камеру Roblox"**: Выравнивание обзора
- **Справка по горячим клавишам**: Встроенная информация

## ⌨️ Горячие клавиши

### 🔵 F7 - Быстрой запуск/стоп макроса
- **Функция**: Включает/отключает воспроизведение
- **Особенности**:
  - Работает глобально (даже когда приложение свернуто)
  - Первое нажатие - запуск
  - Второе нажатие - стоп
- **Используйте когда**: Играете и не хотите разворачивать приложение

### 🔴 F8 - Экстренная остановка (Panic Button)
- **Функция**: Немедленно останавливает все действия
- **Особенности**:
  - Работает глобально
  - Очень быстрое срабатывание
  - Останавливает как запись, так и воспроизведение
- **Используйте когда**: 
  - Что-то пошло не так
  - Нужно срочно остановить макрос
  - Произошла критическая ошибка

## 💾 Сохранение и управление профилями

### 📁 Автоматическое сохранение
- **Где сохраняется**: %APPDATA%\RobloxMacro\Profiles\
- **Формат**: JSON
- **Содержит**:
  - Название макроса
  - Игровой режим
  - Описание
  - Все действия
  - Параметры воспроизведения

### 📝 Экспорт/Импорт
- **Экспорт**: Сохранить макрос в файл (JSON)
- **Импорт**: Загрузить макрос из файла
- **Совместимость**: Файлы совместимы между версиями

### 🏷️ Профили
- **Создать профиль**: Сохранить текущий макрос
- **Загрузить профиль**: Открыть сохраненный макрос
- **Удалить профиль**: Удалить ненужный макрос
- **Список профилей**: Просмотреть все сохраненные макросы

## 🔧 Специальные функции

### 📷 Настройка камеры Roblox
- **Что это делает**: Автоматически выравнивает обзор камеры в Roblox
- **Как работает**:
  1. Нажимает правую кнопку мыши
  2. Перемещает курсор в центр экрана
  3. Отпускает кнопку мыши
  4. Нажимает клавишу 'O' для сброса
- **Зачем нужна**: Обеспечивает стабильность макроса при каждом запуске
- **Когда использовать**: Перед каждой новой игровой сессией

### 🎬 Предпросмотр макроса
- **Функция**: Просмотр всех действий перед воспроизведением
- **Показывает**:
  - Список всех действий
  - Координаты и таймеры
  - Количество действий
  - Расчетное время выполнения

## 🎯 Использование для разных режимов

### Story Mode - Базовое использование
```
Тип действий: Клики, простые комбинации
Количество повторений: 5-20
Тип повтора: Множественные
```

### Legends Stage - Продвинутое использование
```
Тип действий: Комбинации клавиш, быстрые клики
Количество повторений: 20-100
Тип повтора: Множественные с интервалами
```

### Infinity Mode - Специальное использование
```
Тип действий: Основные действия, короткие комбинации
Количество повторений: Бесконечно (∞)
Тип повтора: Непрерывный цикл
```

### Raid - Комплексное использование
```
Тип действий: Сложные комбинации, стратегические действия
Количество повторений: 1-5
Тип повтора: Минимальное количество
```

### Challenge - Экстремальное использование
```
Тип действий: Идеальные комбинации, максимальная точность
Количество повторений: 1-3
Тип повтора: Без погрешностей
```
